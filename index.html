❓ Como Usar

🎯 Navegação

• Use os botões ⬅️ ➡️ para navegar entre os itens

• Clique no menu suspenso para escolher o que estudar

• Use as setas do teclado para navegar (se ativado)

🎮 Jogos

• Escolha "Jogos" no menu para acessar atividades interativas

• Jogo da Memória: encontre os pares iguais

• Jogo de Cores: clique na cor solicitada

• Contagem: conte os objetos na tela

⚙️ Configurações

• Alto Contraste: melhora a visibilidade

• Texto Maior: aumenta o tamanho das letras

• Navegação por Teclado: permite usar as setas

🏆 Conquistas

• Complete atividades para desbloquear conquistas

• Seu progresso é salvo automaticamente

• Visite seu perfil para ver estatísticas

🎨 Personalização

• Escolha seu avatar favorito no perfil

• Ajuste o tamanho dos emojis nas configurações

• Ative o modo de alto contraste se necessário

ℹ️ Sobre o Projeto

📚 Plataforma de Alfabetização TEA

Uma ferramenta educacional desenvolvida especialmente para crianças no espectro autista, focando em métodos visuais e interativos de aprendizagem.

🎯 Objetivos

  • Promover a alfabetização de forma inclusiva
  • Desenvolver habilidades cognitivas
  • Estimular a comunicação
  • Criar um ambiente de aprendizado seguro

✨ Características

  • Interface amigável e acessível
  • Conteúdo educativo diversificado
  • Jogos interativos
  • Sistema de conquistas motivacional
  • Configurações de acessibilidade

🛠️ Tecnologias

HTML5, CSS3, JavaScript (ES6+)

Design Responsivo, PWA Ready

❤️ Compromisso

Este projeto é desenvolvido com amor e dedicação para promover a inclusão educacional e apoiar o desenvolvimento de crianças autistas.
